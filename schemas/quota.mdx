title: Quota

This section outlines the key attributes of the quota's API object. This object can be utilized through the [API](/api-reference/api), [CLI](/api-reference/cli) and the console's apply functionality.

**id** `string`

**kind** `string`

**version** `number`

**description** `string`

**name** `string`

Max Length: 64

**unit** `string`

**dimensions** `object`

**max** `number`

**created** `string`

Format: date-time

**lastModified** `string`

Format: date-time

**current** `number`

**origin** `string`

Default: default

Possible enum values:

*   default
    
*   builtin
    

**links** `array of objects` - [See Below](#attribute-links)
