---
title: Cloud Account
---

This section outlines the key attributes of the cloud account's API object. This object can be utilized through the [API](/api-reference/api), [CLI](/api-reference/cli) and the console's apply functionality.

**id** `string`

**name** `string`

Max Length: 64

**kind** `string`

**version** `integer`

**description** `string`

Max Length: 250

**tags** `object`

**created** `string`

Format: date-time

**lastModified** `string`

Format: date-time

**links** `array of objects` - [See Below](#attribute-links)

**provider** `string`

Possible enum values:

*   aws
    
*   gcp
    
*   azure
    
*   ngs
    

**data** `object` - [See Below](#attribute-data)

**status** `object` - [See Below](#attribute-status)

links[](/schemas/cloud-account#attribute-links)
---------------------------------------------------------------------------

**rel** `string`

**href** `string`

data[](/schemas/cloud-account#attribute-data)
-------------------------------------------------------------------------

The `data` object's structure is determined by the value of the `provider` field. The following providers are supported, with each requiring a unique configuration schema:

*   [gcp](#attribute-data-gcp)
    
*   [aws](#attribute-data-aws)
    
*   [azure](#attribute-data-azure)
    
*   [ngs](#attribute-data-ngs)
    

Depending on the `provider`, the data object must adhere to the specific configuration schema as outlined below.

### gcp[](/schemas/cloud-account#attribute-data-gcp)

**projectId** `string`

Max Length: 30

### aws[](/schemas/cloud-account#attribute-data-aws)

**roleArn** `string`

### azure[](/schemas/cloud-account#attribute-data-azure)

**secretLink** `string`

### ngs[](/schemas/cloud-account#attribute-data-ngs)

**secretLink** `string`

status[](/schemas/cloud-account#attribute-status)
-----------------------------------------------------------------------------

**usable** `boolean`

**lastChecked** `string`

Format: date-time

**lastError** `string`
