title: Secret
---

This section outlines the key attributes of the secret's API object. This object can be utilized through the [API](/api-reference/api), [CLI](/api-reference/cli) and the console's apply functionality.

**id** `string`

**name** `string`

Max Length: 64

**kind** `string`

**version** `integer`

**description** `string`

Max Length: 250

**tags** `object`

**created** `string`

Format: date-time

**lastModified** `string`

Format: date-time

**links** `array of objects` - [See Below](#attribute-links)

**type** `string`

Possible enum values:

*   opaque
    
*   tls
    
*   gcp
    
*   aws
    
*   ecr
    
*   userpass
    
*   keypair
    
*   azure-sdk
    
*   azure-connector
    
*   docker
    
*   dictionary
    
*   nats-account
    

**data** `object` - [See Below](#attribute-data)

links[](/schemas/secret#attribute-links)
---------------------------------------------------------------------

**rel** `string`

**href** `string`

data[](/schemas/secret#attribute-data)
-------------------------------------------------------------------

The `data` object's structure is determined by the value of the `type` field. The following types are supported, with each requiring a unique configuration schema:

*   [opaque](#attribute-data-opaque)
    
*   [tls](#attribute-data-tls)
    
*   gcp `string`
    
*   [aws](#attribute-data-aws)
    
*   [ecr](#attribute-data-ecr)
    
*   docker `string`
    
*   [userpass](#attribute-data-userpass)
    
*   [keypair](#attribute-data-keypair)
    
*   dictionary `object`
    
*   azure-sdk `string`
    
*   [azure-connector](#attribute-data-azure-connector)
    
*   [nats-account](#attribute-data-nats-account)
    

Depending on the `type`, the data object must adhere to the specific configuration schema as outlined below.

### opaque[](/schemas/secret#attribute-data-opaque)

**payload** `string`

**encoding** `string`

Default: plain

Possible enum values:

*   plain
    
*   base64
    

### tls[](/schemas/secret#attribute-data-tls)

**key** `string`

**cert** `string`

**chain** `string`

### aws[](/schemas/secret#attribute-data-aws)

**accessKey** `string`

**secretKey** `string`

**roleArn** `string`

**externalId** `string`

### ecr[](/schemas/secret#attribute-data-ecr)

**accessKey** `string`

**secretKey** `string`

**roleArn** `string`

**externalId** `string`

**repos** `array of strings`

### userpass[](/schemas/secret#attribute-data-userpass)

**username** `string`

**password** `string`

**encoding** `string`

Default: plain

Possible enum values:

*   plain
    
*   base64
    

### keypair[](/schemas/secret#attribute-data-keypair)

**secretKey** `string`

**publicKey** `string`

**passphrase** `string`

### azure-connector[](/schemas/secret#attribute-data-azure-connector)

**url** `string`

Url to the iam-broker Azure connection

**code** `string`

Access code to call the function

### nats-account[](/schemas/secret#attribute-data-nats-account)

**accountId** `string`

**privateKey** `string`
